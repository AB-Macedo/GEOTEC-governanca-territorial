# Aula 08 - Indicador de Sobreposição entre CARs

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| `01_importar_dados.ipynb` | Script para importar os dados do CAR no banco local |
| `02_car_sobreposicao.ipynb` | Notebook principal com o cálculo do indicador |

## Pré-requisitos

- PostgreSQL com PostGIS
- Banco de dados `geotec`
- Dados do CAR (PI) 

## Como usar

1. Execute `01_importar_dados.ipynb` para carregar os dados
2. Execute `02_car_sobreposicao.ipynb` para calcular o indicador
